## Description

SWAPI project created with NestJS including filtering and caching.

## Documentation

You can find swagger documentation on BASE_URL/docs endpoint by default [http://localhost:3000/docs](http://localhost:3000/docs).

## Project setup

### Using docker

```bash
$ docker-compose up
```

### Using pnpm

#### Project setup

```bash
$ pnpm install
```

#### Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

#### Run tests

```bash
# unit tests
$ pnpm run test

# test coverage
$ pnpm run test:cov
```
